# Build Steps (maven, gradle, ant)   
1.Compile  
2.Package   
3.Document
## compile
`javac MyClass.java`
## run
`java MyClass`
## package
`jar cf MyApp.jar MyClass.class Service1.class Service2.class ...`  
##run
`java -jar MyApp.jar`
## document
`javadoc -d doc myclass.java`
